# SimpanAja Platform - Solusi Pintar Pengelolaan Finansial

**SimpanAja** adalah aplikasi manajemen finansial berbasis web yang dirancang untuk membantu pengguna melacak, menganalisis, dan mengoptimalkan pengeluaran serta tabungan mereka dengan mudah dan interaktif. Platform ini menyediakan antarmuka modern dan visualisasi data yang komprehensif bagi individu maupun bisnis skala kecil.

---

## 🚀 Fitur Utama

1. **Dashboard Analitik Keuangan**
   Menyajikan visualisasi data yang interaktif untuk memantau arus kas, pengeluaran bulanan, dan pencapaian target tabungan secara real-time.

2. **Manajemen Transaksi**
   Fitur yang memudahkan pencatatan transaksi masuk dan keluar harian, lengkap dengan kategorisasi pengeluaran dan pemasukan untuk pelaporan yang lebih terstruktur.

3. **Perencanaan dan Target Tabungan**
   Membantu pengguna menetapkan dan melacak progres dari tujuan keuangan spesifik dengan sistem notifikasi dan indikator pencapaian yang mudah dipahami.

4. **Antarmuka Modern dan Responsif**
   Dibangun dengan React dan Tailwind CSS untuk memberikan pengalaman pengguna yang cepat, mulus, dan dapat diakses dengan baik di berbagai perangkat.

---

## 💻 Prasyarat Sistem

Sebelum menjalankan aplikasi, pastikan sistem Anda memenuhi prasyarat berikut:
* **Runtime**: [Node.js](https://nodejs.org/) (Versi 18.0.0 atau yang lebih baru)
* **Package Manager**: NPM (bawaan Node.js, Versi 9.0.0 atau lebih baru)
* **Browser**: Browser web modern apa saja (Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, dll.)

## 🛠️ Petunjuk Instalasi dan Konfigurasi

Ikuti langkah-langkah di bawah ini untuk memasang dan menjalankan aplikasi SimpanAja di komputer lokal Anda:

1. **Masuk ke Direktori Proyek**:
   ```bash
   cd SimpanAja
   ```

2. **Instal Dependensi**:
   Lakukan instalasi modul yang dibutuhkan:
   ```bash
   npm install
   ```

3. **Jalankan Server Pengembang**:
   Jalankan perintah berikut untuk memulai server pengembang Vite:
   ```bash
   npm run dev
   ```
   Aplikasi akan berjalan secara otomatis dan Anda dapat mengaksesnya melalui tautan lokal yang ditampilkan di terminal (biasanya `http://localhost:5173`).

---

## 📖 Cara Penggunaan

Setelah server berhasil dijalankan, buka peramban Anda dan kunjungi URL yang diberikan oleh Vite (misalnya `http://localhost:5173`).

* **Dashboard Utama**: Menampilkan ringkasan keuangan dan grafik analitik dari data transaksi Anda.
* **Menu Navigasi**: Gunakan navigasi yang tersedia untuk berpindah antar halaman seperti Daftar Transaksi, Tambah Transaksi, dan Pengaturan Profil.
* **Manajemen Data**: Anda dapat mulai memasukkan data keuangan simulasi untuk menguji fungsionalitas pencatatan dan analitik pada aplikasi ini.
